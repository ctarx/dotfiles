## Install

I use GNU Stow [https://www.gnu.org/software/stow](https://www.gnu.org/software/stow)

```shell
cd $HOME
git clone https://github.com/ctarx/omarchy-dotfiles.git ~/.dotfiles
cd ~/.dotfiles
stow home && stow config
```
